## 1. 🏗️ 项目全景与蓝图 (Project Blueprint)

* **项目名称**: Pulsar (脉冲星)
* **项目代号**: Pulsar Redux
* **核心价值**: 消除“切换”与“执行”之间的微秒级阻力。
* **版本目标**: 构建一个逻辑严格分离、执行绝对稳定、交互符合直觉的系统级效率轮盘。

### 核心架构变革 (Architecture Evolution)

| 特性 | 旧版 Pulsar (v1.x) | **Pulsar Redux (v2.0)** |
| --- | --- | --- |
| **交互逻辑** | 单一混合逻辑 | **双模态物理隔离** (Launcher vs Smart Command) |
| **判定算法** | 简单的碰撞检测 | **无限扇区极坐标算法** (Infinite Sector Selection) |
| **数据模型** | 扁平结构 (`GridItem`) | **多态继承结构** (`GridItemBase` -> `LauncherItem`/`CommandItem`) |
| **序列化** | TOML (Tomlyn) | **JSON** (`System.Text.Json` + Polymorphism) |
| **配置管理** | 单一列表 | **顶层分流** (SwitcherSlots + CommandLayer) |

## 3. 📜 技术契约与规范 (Technical Contracts)

### 3.1 数据序列化规范 (Data Serialization)

* **JSON 多态**: 必须使用 `System.Text.Json` 的 `[JsonDerivedType]` 特性处理多态对象。
* 基类: `GridItemBase`
* 标识符: `typeDiscriminator` ("launcher", "command")
* **配置迁移**: 废弃 TOML，全面转向 JSON，以原生支持复杂的对象继承结构。

### 3.2 MVVM 与 UI 绑定 (MVVM & Binding)

* **显式属性定义**: 对于高频布局属性（如 `X`, `Y`），**必须**在 ViewModel 中显式手写属性定义，严禁完全依赖 Source Generator，避免 Binding 失效。
* **状态封装原则**: ViewModel 的核心状态（如 `IconGlyph`）若设有 `private set`，**必须**提供公开的方法（如 `LoadIconData`）供外部调用，严禁外部直接修改状态属性。
* **UI 样式作用域**: 在 `UserControl.Style` 中，`Setter` 无法通过 `TargetName` 访问控件内部元素。**必须**将可见性触发逻辑（Triggers）移入内部元素（如 `Image`, `TextBlock`）自身的 `Style` 定义中。

### 3.3 双模态交互 (Dual-Mode Interaction)

* **模式一 (Launcher)**: `Ctrl + Shift + Q` -> 强制加载 `SwitcherSlots`。
* **模式二 (Smart Command)**: `Ctrl + Q` -> 上下文感知加载 Profile。

### 3.4 智能执行 (Smart Execution)

* **禁止隐式解析**: `CommandService` **严禁**对命令字符串进行 Split 操作。
* **强类型传递**: 参数必须通过 `CommandItem.Arguments` 显式传递给 `Process.StartInfo`。

## 5. 🛠️ 故障排查备忘 (Troubleshooting Memory)

* **Q: 轮盘所有图标显示为方形豆腐块?**
* **A**: 缺少 Windows 11 的 Segoe Fluent Icons 字体。**Fix**: 在 XAML 中添加 `Segoe MDL2 Assets` 作为回退字体。
* **Q: 编译报错 "无法在 Style Setter 上设置 TargetName 属性"?**
* **A**: 试图在 UserControl 的外层 Style 中控制内部命名元素 (`x:Name`) 的属性。**Fix**: 将 Trigger 逻辑下沉到对应元素的 `<Style>` 内部。
* **Q: 编译报错 "属性 set 访问器不可访问"?**
* **A**: ViewModel 属性被设为 `private set`，但外部类试图赋值。**Fix**: 改为调用 ViewModel 暴露的公共方法（如 `LoadIconData`）来更新状态。
