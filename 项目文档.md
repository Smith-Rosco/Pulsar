* **项目名称**: Pulsar (脉冲星)
* **项目代号**: Pulsar Redux
* **核心价值**: 消除“切换”与“执行”之间的微秒级阻力。
* **版本目标**: 构建一个逻辑严格分离、执行绝对稳定、交互符合直觉的系统级效率轮盘。

| 特性 | 旧版 Pulsar (v1.x) | **Pulsar Redux (v2.0)** |
| --- | --- | --- |
| **交互逻辑** | 单一混合逻辑 | **双模态物理隔离** (Launcher vs Smart Command) |
| **判定算法** | 简单的碰撞检测 | **无限扇区极坐标算法** (Infinite Sector Selection) |
| **窗口管理** | 标准 Show/Hide | **驻留幽灵模式** (Resident Ghost Mode - Opacity Control) |
| **执行逻辑** | 仅启动或仅切换 | **幂等执行 (Idempotency)**: 自动判定 Switch ➔ Fail ➔ Launch |

| **渲染机制** | 依赖布局系统 | **预热与强制刷新** (Pre-flight Layout Update) |

## 3. 📜 技术契约与规范 (Technical Contracts)

### 3.1 独立组件架构 (Independent Component Architecture)

* **去耦合**: 基础 UI 组件（如 `JellyOrb`）**严禁**依赖隐式的 `DataContext` 继承。
* **显式驱动**: 必须使用 `DependencyProperty` 定义输入接口。
* **动态资源**: 必须使用 `DynamicResource` 引用画笔。

### 3.2 跨线程与原生资源 (Interop & Thread Safety) [New]

* **后台 UI 对象冻结**: 在非 UI 线程（如 `Task.Run`）创建的 `BitmapSource` 或 `Freezable` 对象，必须在返回前调用 `.Freeze()`，否则 UI 线程访问时会抛出异常。
* **原生图标提取**: 使用 `SHGetFileInfo` 提取图标时，必须通过 `Imaging.CreateBitmapSourceFromHIcon` 转换，并立即 `DestroyIcon` 释放原生句柄防止内存泄漏。
* **默认资源**: **严禁**依赖本地物理文件（如 `.png`）作为默认图标。必须使用 XAML 绘图或 Emoji/Unicode 字符作为 Fallback，以避免 `IOException`。

### 3.3 瞬态窗口管理 (Transient Window Management) [New]

* **主题注入**: 新创建的 `Window` (Dialogs) 默认不包含 App 级资源。必须在 `ShowDialog()` 之前调用 `ThemeManager.ApplyTheme(dialog, currentTheme)` 手动注入当前主题字典。
* **所有者绑定**: 必须显式设置 `dialog.Owner` 为当前激活窗口，防止弹窗在 Alt+Tab 时丢失层级。
* **深色模式适配**: `ListBox` 或 `ListView` 内部的 `DataTemplate` 必须显式绑定 `Foreground="{DynamicResource Theme.Window.Foreground}"`，否则系统默认样式可能会在深色背景下显示黑色文字。

### 3.4 驻留窗口模式 (Resident Window Pattern)

* **生命周期**: 主窗口永不销毁。
* **数据消毒**: `Dismiss` 时调用 `ViewModel.ClearVisuals()`。

### 3.5 进程命名规范 (Process Naming Convention)

* **无后缀**: `WindowService` 返回的 `ProcessName` 不包含 `.exe`。
* **数据清洗**: 在设置界面保存或选择进程时，必须自动移除 `.exe` 后缀并转为小写。

## 4. 🧩 已实现功能地图 (Implemented Feature Map)

| 模块 | 功能点 | 实现逻辑摘要 |

| **Logic** | **智能启停 (Smart Launch)** | `CommandService` 逻辑升级：优先尝试 `SwitchToProcess`；若返回 false，则回退执行 `Process.Start(ExePath)`。实现了“无论状态如何，结果都是程序呈现在面前”的幂等性。 |

## 5. 🛠️ 故障排查备忘 (Troubleshooting Memory)

* **Q: 打开选择器弹窗时崩溃，提示 IOException?**
* **A**: 代码中引用了不存在的 `Resources/default.png`。**Fix**: 移除图片引用，改用 XAML 代码绘制或 Unicode 字符作为默认图标。
* **Q: 弹窗背景是白色的，或者文字看不清（样式丢失）?**
* **A**: 新窗口没有继承 App.xaml 的资源。**Fix**: 在 `ShowDialog` 前调用 `ThemeManager.ApplyTheme` 注入当前主题资源。
* **Q: 深色模式下，列表文字是黑色的？**
* **A**: `ListBoxItem` 的默认样式在某些系统主题下 Foreground 为黑。**Fix**: 在 DataTemplate 中的 TextBlock 显式绑定 `Foreground="{DynamicResource Theme.Window.Foreground}"`。
* **Q: 轮盘唤起时有短暂的旧图标残影?**
* **A**: **Fix**: 关闭时 `ClearVisuals()`；开启时先 `UpdateLayout()` 再 `Opacity=1`。
